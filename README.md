# Ιστοσελίδα Εγγραφής Δοτών Αίματος

Αυτό το έργο είναι μια απλή, ευέλικτη εφαρμογή ιστού σχεδιασμένη για να επιτρέπει στους χρήστες να εγγράφονται ως δότες αίματος. Η ιστοσελίδα καταγράφει το ονοματεπώνυμο, τον αριθμό τηλεφώνου, το email και την ημερομηνία και ώρα εγγραφής του δότη και στη συνέχεια υποβάλλει αυτά τα δεδομένα σε ένα API.

## Χαρακτηριστικά

- **Ευέλικτος Σχεδιασμός**: Η ιστοσελίδα είναι φιλική προς κινητές συσκευές και προσαρμόζει τη διάταξή της σε διάφορα μεγέθη οθονών.
- **Ενσωμάτωση Λογότυπου**: Η ιστοσελίδα περιλαμβάνει ένα προσαρμόσιμο λογότυπο στην κορυφή της φόρμας εγγραφής.
- **Καταγραφή Ημερομηνίας και Ώρας**: Η φόρμα εγγραφής καταγράφει αυτόματα την τρέχουσα ημερομηνία και ώρα όταν ο χρήστης υποβάλλει τις πληροφορίες του.
- **Διαφανές Favicon**: Η ιστοσελίδα χρησιμοποιεί ένα favicon με διαφανές φόντο που ενσωματώνεται άψογα σε οποιοδήποτε θέμα του προγράμματος περιήγησης.
- **Ενσωμάτωση API**: Τα δεδομένα της φόρμας υποβάλλονται σε ένα API μέσω ενός αιτήματος `fetch` στη JavaScript, εξασφαλίζοντας ομαλή και σύγχρονη διαχείριση της φόρμας.
- **Μήνυμα Ευχαριστίας**: Μετά την υποβολή της φόρμας, οι χρήστες λαμβάνουν ένα μήνυμα ευχαριστίας με πρόσθετες πληροφορίες για τη σημασία της εγγραφής τους.

## Δομή Έργου

- **index.html**: Το κύριο αρχείο HTML που περιέχει τη φόρμα εγγραφής.
- **style.css**: Το αρχείο CSS για το στυλ της ιστοσελίδας.
- **logo.png**: Η εικόνα του λογότυπου που εμφανίζεται στην ιστοσελίδα.
- **favicon.ico**: Favicon με διαφανές φόντο.
- **api.php**: Το αρχείο PHP που διαχειρίζεται την υποβολή της φόρμας και αποθηκεύει τα δεδομένα στη βάση δεδομένων.
- **cors.php**: Το αρχείο PHP που διαχειρίζεται τα CORS headers.
- **db_config.php**: Το αρχείο ρύθμισης παραμέτρων της βάσης δεδομένων.
- **list_registrations.php**: Λίστα Δοτών, με προστασία κωδικόυ line:8@list_registrations.php

## Ρυθμίστε τη Βάση Δεδομένων:

Ρυθμίστε τη βάση δεδομένων MySQL χρησιμοποιώντας τα διαπιστευτήρια στο db_config.php.
Βεβαιωθείτε ότι το αρχείο api.php είναι σωστά συνδεδεμένο με τη βάση δεδομένων MySQL.

## Χρήση
Εγγραφή: Οι χρήστες συμπληρώνουν τη φόρμα με το ονοματεπώνυμό τους, τον αριθμό τηλεφώνου και το email τους. Η φόρμα καταγράφει αυτόματα την ημερομηνία και ώρα εγγραφής.
Υποβολή Φόρμας: Με την υποβολή της φόρμας, τα δεδομένα αποστέλλονται στο αρχείο api.php, το οποίο επεξεργάζεται και αποθηκεύει τις πληροφορίες στη βάση δεδομένων.
Επιβεβαίωση: Ένα μήνυμα ευχαριστίας εμφανίζεται για να επιβεβαιώσει την επιτυχημένη εγγραφή.
